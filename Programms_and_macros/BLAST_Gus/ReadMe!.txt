Процедура для автоматизация процедуры отправки запросов и получения списка родственных последовательностей в BLAST.
Запускаемый файл "blast.exe"

Исходные последовательности, к которым ищем родственников в файле input_sequences.fasta.
Название и путь файла можно изменить, вписав соответствующую информацию в файл settings.json в значение ключа "input_seq".
Следует учеть, что в пути к файлу не должно быть символа "\"! Вместо него необходимо использовать "/" (особенности синтаксиса Python). 

Количество желаемых выравниваний (производимых в BLAST) для поиска родственников
и количество топовых позиций по максимально близкому родству
среди найденных последовательностей - "alignments" и  "hitlist_size", соответственно.
