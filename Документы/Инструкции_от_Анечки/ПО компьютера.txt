На данном компьютере установлены

1. Miniconda с виртуальными средами: 
	Telebot - для работы с телеграм-ботом и его запуском;
	Docking - для работы с докингом и его запуском.
2. WSL for Windows (эмулятор Linux) - для запуска докинга, который работает лишь на ОС Linux.
3. Autodock Tools - позволяет создать grid-box для докинга.
4. Pymol - визуализация и редактирование молекул для процесса докинга.
